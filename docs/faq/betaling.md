# Hvordan fungerer betaling?

Du har 3 alternativer for å betale for tjenesten:

## 1. Kredittkort

Du kan betale med kredittkort ved å legge inn kortinformasjonen din på betalingsportalen. Vi aksepterer de fleste store
kredittkort, og betalingen behandles umiddelbart.

## 2. PayPal

Du kan også betale via PayPal. Velg PayPal som betalingsmetode, og du vil bli omdirigert til PayPal-nettstedet for å
fullføre betalingen.
