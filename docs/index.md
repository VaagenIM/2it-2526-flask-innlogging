## Installasjon

Bruk følgende `.env`-fil for å sette opp miljøvariablene dine:

```env
# Database
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=yourpassword
DB_NAME=yourdatabase

# Server
PORT=3000
```
